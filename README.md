# 天枢桥检 — 桥梁智能巡检与AI定损系统

> 🌉 基于 **YOLOv8 + Jetson Orin NX + 无人机** 的桥梁缺陷智能检测平台

## 📖 项目简介

天枢桥检是一套完整的桥梁智能巡检解决方案，采用无人机搭载**双目摄像头 + 红外热成像 + CBAM注意力机制 + 超声波传感器**，对桥梁进行全自动巡检与AI定损。

### 核心能力
- 🎯 **7类缺陷自动识别**：裂缝、钢筋外露/锈蚀、混凝土剥落、螺栓缺失、涂层脱落、水渍等
- 🤖 **YOLOv8 检测模型**：实测精度 **96.8%**，推理速度 ~25 FPS
- 🚁 **自主巡航**：巡航速度 30km/h，续航约 **65分钟**
- 🌡️ **红外热成像**：深度融合热图数据辅助定损
- 🧠 **LLM 决策引擎**：自动生成处置方案（P0-P2优先级）
- ☁️ **云控平台**：机巢 + 云端管理三层架构
